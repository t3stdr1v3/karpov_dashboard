# karpov_dashboard
В этом проекте собраны дашборды и запросы sql по датасету из курса Карпова "Симулятор SQL". Запросы написано на PosgreSQL 

### Для интерактивного взаимодействия с дашбордами перейдите по ссылке 
https://redash.public.karpov.courses/public/dashboards/KMJD0pkgu00A54KElmy8Hrt9zHYIeIjVJvesks4p?org_slug=default

Запросы вы можете найти в файле queries.ipynb. Учитывая формат файла, вы не сможете запустить тот код, но сможете запустить при помощи REDASH на датасете simulator sql.
